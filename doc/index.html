Module deathinventorydrop

A system which makes the player drop their inventory when they die.

Functions

activate () Activates the system, if it has not been disabled in the configuration by setting "deathinventorydrop_activate" to "false".
drop (player) Drops the inventory of the given player.
drop_inventory (player, inventory, list_name) Drops the given list from the given player.

Fields

inventories The list of inventories/list names to drop, they need to be in players inventory, defaults to "main".
split The split mode to use, can either be "random", "single" or "stack", defaults to "random".
velocity The velocity with which the items are dropped, defaults to "7, 10, 7".
